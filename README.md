# AI防犯カメラ Androidアプリ

古いAndroidスマホを玄関・窓・駐車場などに向けて設置し、人の検知、骨格推定、簡易的な不審行動スコアリングを行うMVPアプリです。

## 方針

- 顔認証は使わない
- 個人名や住所などの個人情報を特定しない
- 「この人は誰か」ではなく「過去に不審行動をした人物と似ているか」を扱う
- 最初はMediaPipe Poseとルールベース判定でMVPを作る
- データはまずローカル保存を前提にする

## MVP

1. Androidでカメラを起動する
2. 人が映ったら検知する
3. MediaPipe Poseで骨格ランドマークを取得する
4. 骨格データをローカルに保存する
5. 停止時間、画面内の移動量、滞在時間から怪しさスコアを出す
6. 怪しさスコアが一定以上なら通知・保存する

## Git運用

開発の節目ごとに小さくコミットします。

- `main`: 安定版
- 作業ブランチ: 機能ごとに作成
- コミットメッセージ: 何を変えたかが分かる短い文にする

## 技術方針

MVPはKotlinネイティブAndroidアプリとして作ります。

- カメラ: CameraX
- 骨格推定: MediaPipe Pose Landmarker for Android
- 判定: Kotlin内のルールベース処理
- 保存: ローカル保存
- Python: MVP後の分析・実験・サーバー用途

詳細は[docs/architecture.md](docs/architecture.md)を参照。
